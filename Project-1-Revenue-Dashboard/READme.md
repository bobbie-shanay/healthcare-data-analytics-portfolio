Revenue Dashboard (Power BI)

Overview
This project focuses on analyzing healthcare revenue performance across multiple hospice organizations. The dashboard provides insights into revenue trends, patient volume, and financial performance to support data-driven decision-making.

Tools Used
- Power BI  
- Excel  

Key Metrics
- Total Revenue  
- Revenue by Hospice  
- Revenue Trends by Month  
- Patient Volume  

Features
- Interactive dashboard with slicers for filtering by month  
- Visual breakdown of revenue across different hospice organizations  
- Trend analysis to track changes in revenue over time  

Insights
- Identified variations in revenue performance across hospice providers  
- Highlighted trends in monthly revenue growth  
- Provided visibility into patient volume and its impact on revenue
