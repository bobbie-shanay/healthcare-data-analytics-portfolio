SQL Claims Analysis

Overview
This project uses SQL to analyze healthcare claims data and evaluate financial performance across hospice organizations.

Tools Used
- SQL  
- Excel
- Power BI

Key Metrics
- Total Billed Amount  
- Total Paid Amount  
- Revenue Loss  
- Denial Rate  

Features
- Aggregated claims data to calculate financial metrics  
- Analyzed differences between billed and paid amounts  
- Supported filtering by hospice to compare performance  

Description
SQL queries were used to extract, transform, and analyze claims data. The results were used to evaluate revenue performance and identify gaps between billed and paid amounts.

Insights
- Identified revenue loss across hospice organizations  
- Highlighted differences between billed and paid amounts  
- Provided visibility into denial rates and financial performance
